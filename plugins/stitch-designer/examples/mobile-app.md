# Mobile App Example

```text
Use stitch-designer to create a mobile app home screen for a bilingual language-learning app.

If no project is selected, ask whether to reuse an existing Stitch project or create a new one.

The screen should help learners resume the next lesson, review daily progress, and start a quick practice. Use a warm editorial notebook aesthetic, high contrast, clear 44px+ touch targets, and no marketing hero. Include loading, empty, and streak-complete states in the prompt.
```
