Codex Stitch Designer

Design better Stitch screens from Codex and Claude Code.

A plugin package for Google Stitch MCP that helps generate UI screens, explore variants, critique design output, and prepare frontend implementation handoffs.

Install from GitHub Stitch MCP setup

What it does

Project-aware

Asks whether to reuse an existing Stitch project, create a new one, or work without Stitch.

Context-aware

Adapts prompts for mobile apps, web apps, dashboards, landing pages, games, and RTL UI.

Review-ready

Critiques output for workflow clarity, hierarchy, contrast, accessibility, and implementation risk.

Install

Codex

codex plugin marketplace add https://github.com/alabenkhlifa/codex-stitch-designer.git
codex plugin list --marketplace codex-stitch-designer
codex plugin add stitch-designer@codex-stitch-designer

Claude Code

claude plugin marketplace add alabenkhlifa/codex-stitch-designer
claude plugin install stitch-designer@stitch-designer
